## 실행 방법
소스코드 다운로드
```
https://github.com/DevCHW/spring-restdocs-enum-popup-kotlin.git
```

빌드
```
./gradlew clean build
```

실행
```
java -jar build/libs/spring-restdocs-enum-popup-kotlin-0.0.1-SNAPSHOT.jar
```

## 환경
- Kotlin: 2.1.0
- SpringBoot: 3.4.1
